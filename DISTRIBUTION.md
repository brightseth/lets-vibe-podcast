# Let's Vibe! Distribution Setup

## Hosting Decision

**Recommended: Spotify for Creators (Free)**
- Unlimited storage
- Video podcast support
- 713M monthly active users on Spotify
- Built-in monetization via Spotify Partner Program
- Manual RSS submission to Apple Podcasts required

**Alternative: Transistor ($19/mo)**
- One-click distribution to all platforms
- Better analytics
- Multiple shows on one account

## Setup Steps

### 1. Spotify for Creators Setup
1. Go to [creators.spotify.com](https://creators.spotify.com)
2. Sign in with Spotify account
3. Create new podcast
4. Upload cover art (1400x1400 minimum, square)
5. Fill in show details:
   - Title: Let's Vibe!
   - Description: [write compelling description]
   - Category: Technology > Tech News (or Arts > Design)
   - Language: English
   - Explicit: No (probably)
6. Upload first episode
7. Get RSS feed URL

### 2. Apple Podcasts Connect
1. Go to [podcastsconnect.apple.com](https://podcastsconnect.apple.com)
2. Sign in with Apple ID
3. Click "Add a Show"
4. Select "Add a show with an RSS feed"
5. Paste RSS feed URL from Spotify
6. Click "Validate"
7. Click "Submit"
8. **Wait 24 hours to 2 weeks for approval**

### 3. Other Platforms (Optional)
- Amazon Music / Audible
- Google Podcasts (via RSS)
- YouTube (video version)
- Overcast, Pocket Casts (via RSS)

## Requirements Checklist

### Audio
- [ ] MP3 format
- [ ] 128kbps minimum (192kbps recommended)
- [ ] Mono or stereo
- [ ] Clean audio, no clipping

### Artwork
- [ ] 1400x1400 pixels minimum
- [ ] 3000x3000 pixels recommended
- [ ] Square format
- [ ] JPEG or PNG
- [ ] RGB color space
- [ ] No explicit imagery in artwork

### Metadata
- [ ] Show title
- [ ] Show description (4000 chars max)
- [ ] Author name
- [ ] Category selection
- [ ] Language
- [ ] Episode titles and descriptions
