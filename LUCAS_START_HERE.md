## Agent System

The agent system in `letsvibe-agent/` can help with:

```bash
cd letsvibe-agent
npm install
npm run agent                    # See system status
npm run agent route "your task"  # Route to right agent
```

Agents available:
- **Booking** - Guest research, outreach drafting
- **Research** - Prep, talking points
- **Promotion** - Social, graphics, clips
- **Partnerships** - Sponsors, brand deals
